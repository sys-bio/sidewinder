
SBMLtest: Test project for Sidewinder.

Purpose: 
Test reading SBML models and storing them correctly. Also allows developer to test different ode solvers. Web interface displays most of the SBML objects that were read in with libsbmljs.


Note: due to some of the JavaScripts being located out the SBMLtest root directory, copies of them have been placed in its root (libsbml.js, libsbml.wasm, RK4solver.js
